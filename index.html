Korovan

Founded in 2013, Korovan software is a privately-held company. The very first application was the Snowfall live wallpaper for Android.

Heretofore our main line is Android mobile applications.

As a matter of principle, we don’t implement any components hidden from the user, but try to provide the most clear and smooth user experience.
